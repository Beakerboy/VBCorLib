@remarks

This method is not synonymous with checking the length of a string for zero using the Len method. This method checks if the string parameter points to any kind of string, including an empty string (""). A null string is uninitialized.

Examples

Public Sub Main()
    Dim s As String
    
    s = vbNullString ' redundant, but illistrates null pointer.
    Console.WriteLine "Is Null: {0}", CorString.IsNull(s)
    
    s = ""
    Console.WriteLine "Is Null: {0}", CorString.IsNull(s)
    Console.ReadKey
    
' The example outputs:
' Is Null: True
' Is Null: False
End Sub
